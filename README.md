# Panda Press

- A modern CMS but without composer

## Features

- MVC architecture
- PHP 8.2+
- PSR-4 compatibility
- composer free, beginner friendly
- AdminLTE dashboard
- powerful plugins
- beautiful themes
- open source
- always free
- MIT license
